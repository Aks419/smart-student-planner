# Smart Student Planner

A simple desktop planner app built using Python and Tkinter to help students manage tasks and deadlines.

## ✨ Features
- Add tasks with deadlines
- Set priority levels
- Mark tasks as completed
- Auto-save tasks
- Clean and simple interface

## 🛠 Technologies Used
- Python
- Tkinter (GUI)
- JSON for data storage

## ▶ How to Run

1. Install Python
2. Download the project
3. Run:

   python planner.py
